## 🔧 **개발 환경에서 사용법**

### **1. 정제된 데이터만 사용 (권장)**
```bash
# 바로 백엔드 실행 가능
cd backend
npm install
npm run start:dev
```

### **2. 원본 데이터 활용 (선택사항)**
```bash
# 1. 위 URL에서 데이터 다운로드
# 2. data_ai/raw/ 폴더에 압축 해제
# 3. 정제 스크립트 실행
cd data_ai
python aihub_food_litepack.py
```
